Installing CLI

Requirements

  • Windows 10, Mac ISX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install roukicloud-cli

NPM

$ npm install roukicloud-cli

Yarn

$ yarn install roukicloud-cli
